== Description ==

<b>The Swedish Word Of The Day is an educational and fun plugin for WordPress that every day displays a Swedish word and it's translation into English.</b>

This plugin is provided by swedishwordplugin.com and uses the site to fetch data. Read more at <a href="http://swedishwordplugin.com" target="_blank">Swedishwordplugin.com</a>.


<h2>Swedish Word Plugin Features:</h2>

<ul>
<li>One new Swedish word every Day</li>
<li>Translations into English</li>
<li>Inherit colors and fonts from website</li>
<li>Responsive widget</li>
</ul>

Install the plugin on your WordPress website, activate it and display the Swedish word of the day in widget or in the content using the shortcode [swedish_word]



== Installation ==

 Upload The Swedish Word Of The Day Plugin to your website and activate.

    - Add the widget to your sidebar or any widget area or use the shortcode [swedish_word] on your post or page.
    - You're done.
